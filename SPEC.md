# 求职申请管理看板 - 产品规格书

## 1. Concept & Vision

**核心理念**：帮助大学生在求职季告别混乱的表格和备忘录，用清晰的看板一眼掌控所有申请进度。

**设计关键词**：专注、高效、温暖、不焦虑

**用户痛点**：
- 同时跟进10+家公司，每家进度不同
- 错过笔试/面试截止时间
- 不知道哪些材料还没准备
- 面试进度黑盒子，不知道下一轮在哪
- 拿到offer不知道怎么比较

## 2. Design Language

### 视觉风格
- **风格**：现代极简 + 柔和圆角，带有轻微毛玻璃质感
- **情绪**：干净清爽，像整理好的笔记本，让人平静

### 色彩系统
```
--primary: #6366F1      // 靛蓝色 - 主操作
--success: #10B981      // 翠绿色 - 已完成/通过
--warning: #F59E0B      // 琥珀色 - 提醒/待处理
--danger: #EF4444       // 红色 - 截止日期临近/拒绝
--purple: #8B5CF6       // 紫色 - offer
--gray-50: #F9FAFB      // 背景灰
--gray-100: #F3F4F6
--gray-200: #E5E7EB
--gray-500: #6B7280
--gray-700: #374151
--gray-900: #111827
```

### 看板列定义（6列）
1. **意向中** - 有意向但还未投递的公司
2. **已投递** - 简历已提交，等待回复
3. **笔试中** - 收到笔试邀请
4. **面试中** - 面试阶段（含一面、二面、HR面等）
5. **已结束** - 拿到offer 或 被拒绝
6. **offer比较** - 收到的offer放这里对比

### 字体
- 主字体：'Inter', -apple-system, sans-serif
- 中文：'Noto Sans SC', sans-serif

## 4. Features & Interactions

### 核心功能

#### 4.1 申请卡片 CRUD
- **新增申请**：点击「+新申请」→ 填写公司、职位、链接、截止日期、所需材料清单
- **拖拽移动**：卡片可在列间自由拖拽改变状态
- **快速编辑**：点击卡片右上角「···」→ 编辑/删除/归档
- **详情展开**：点击卡片展开详情侧边栏

#### 4.2 截止日期管理
- 卡片左上角显示距离截止的天数
- 颜色编码：绿色(>3天) → 黄色(1-3天) → 红色(已过期/今天)
- 日历视图按日期排列所有截止节点

#### 4.3 材料准备清单
每张卡片可配置材料清单：
- [ ] 简历
- [ ] cover letter
- [ ] 成绩单
- [ ] 作品集
- [ ] 身份证/证件照
- 勾选后显示完成进度 (3/5)

#### 4.4 面试进度追踪
面试阶段可细分为：
- 电话一面
- 在线测评/笔试
- 技术面试
- 现场面试/终面
- HR面
- OFFER沟通

#### 4.5 Offer 比较（多方对比视图）
并排展示多个offer的关键信息：
| 公司 | 薪资 | 福利 | 发展 | 地点 | 截止 |
|------|------|------|------|------|------|
表格形式一目了然

#### 4.6 统计仪表盘
- 申请总数 / 已投递 / 面试中 / offer数
- 各阶段转化漏斗图
- 申请趋势折线图（每周申请数）

### 交互细节
- 拖拽卡片时：卡片轻微放大、阴影加深、原位置显示虚线占位
- 悬停卡片：显示快捷操作按钮
- 截止日期临近：卡片边框闪烁警告色
- 新增/删除：平滑动画过渡

## 5. Component Inventory

### 5.1 KanbanColumn
- 列标题 + 计数徽章
- 可滚动的卡片容器
- 拖拽悬停时边框高亮

- States: default / hover / dragging / overdue / new-offer
- 显示：公司logo(首字母) / 公司名 / 职位 / 截止日期 / 材料进度 / 当前阶段

### 5.3 AddApplicationModal
- 步骤表单：基本信息 → 材料清单 → 截止日期
- 自动补全公司名称（历史记录）

### 5.4 CardDetailSidebar
- 右侧滑出面板
- 完整材料清单编辑
- 面试时间线记录
- 备注/链接区域

### 5.5 OfferComparisonTable
- 可添加/移除对比列
- 高亮最优项

### 5.6 StatisticsPanel
- 数字卡片展示关键指标
- 迷你漏斗图
- 趋势折线图

## 6. Technical Approach

### 技术栈
- **前端**：HTML5 + CSS3 + Vanilla JavaScript
- **存储**：LocalStorage（数据持久化，无需后端）
- **样式**：原生CSS变量 + Flexbox/Grid

### 数据模型
```javascript
Application {
  id: string,
  company: string,
  position: string,
  link: string,
  stage: 'intention' | 'applied' | 'test' | 'interview' | 'ended' | 'offer',
  deadline: Date,
  materials: [{ name: string, done: boolean }],
  interviewRounds: [{ name: string, date: Date, status: 'pending' | 'done' }],
  salary?: string,
  notes?: string,
  createdAt: Date,
  updatedAt: Date,
  outcome?: 'offer' | 'rejected'
}
```

### 响应式策略
- 桌面(>1200px)：完整6列看板
- 平板(768-1200px)：水平滚动看板
- 手机(<768px)：单列视图 + 顶部Tab切换列
